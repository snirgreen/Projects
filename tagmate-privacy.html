Privacy Policy – TagMate

Last updated: October 29, 2025

TagMate is an internal Chrome extension that enables tagging and annotation workflows on designated internal web pages. This policy explains what data the extension uses and how it is handled.

Data Collection

TagMate does not collect or transmit personal user data. The extension stores only the minimum technical data required to provide its functionality, which may include:

  • Tag identifiers and labels
  • Basic page context (e.g., URL) for associating tags with pages the user explicitly interacts with
  • User interface preferences and feature settings

Data Usage

Any stored data is used solely to provide tagging features on authorized pages. Processing occurs locally in the user’s browser or within organization-controlled services, where applicable. Data is not used for advertising or profiling.

Data Sharing

TagMate does not share data with third parties.

Permissions

TagMate requests the following Chrome permissions strictly to enable its core functionality:

  • activeTab – Identify the current tab and inject the tagging UI only after explicit user activation.
  • tabs – Obtain the active tab ID/URL to associate and restore tags for that page.
  • scripting – Programmatically inject packaged content scripts that render the tagging interface on permitted pages.
  • storage – Save tag metadata and user preferences locally (and optionally sync within the browser ecosystem).
  • Host permissions – Limit operation to designated internal domains (e.g., https://internal.example.org/*), preventing access on unrelated sites.

Remote Code

TagMate does not execute remote code. All executable code is bundled within the extension. Network requests are limited to retrieving non-executable data (such as JSON) from organization-controlled endpoints.

Retention

Data is retained only as long as needed for tagging workflows and may be removed according to organizational policy or upon user/admin request.

Access

This extension is intended for authorized users within the organization and is not available for public installation.

Contact

For privacy-related questions or requests, contact: tagmate-privacy@example.org.

If this policy is hosted on an internal site, ensure it is accessible to reviewers during extension review.
